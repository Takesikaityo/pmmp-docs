Server関連

ここに書いてあるものを使うには use pocketmine\Server; が必要です。
関数名 説明
getServer() サーバーのオブジェクト関連を取得する。
これはまだ作成途中です。GithubでPullリクエストをして項目を増やしていきましょう！
関数名 パラメータ 返り値のタイプ 説明
・基本
getName() 不要 string サーバーのソフト名を返します
getPocketMineVersion() 不要 string PMMPのバージョンを返します
getCodename() 不要 string サーバーのコードネームを返します
getVersion() 不要 string 対応しているMCPEのバージョンを返します
getApiVersion() 不要 なし PMMPのAPIバージョンを返します
reload() 不要 なし 再読み込み
shutdown() 不要 なし サーバーを停止
forceShutdown() 不要 なし サーバーを強制停止
・フォルダーパス
getFilePath() 不要 なし サーバーのフォルダーパスを返します
getDataPath() 不要 なし データーのフォルダーパスを返します
getPluginPath() 不要 なし Pluginのフォルダーパスを返します
・その他
getMotd() 不要 string サーバー名取得
hasWhitelist() 不要 array? ホワイトリストに追加済みのプレイヤーを取得
getIp() 不要 string ipを取得
getPort() 不要 int Portを取得
getServerUniqueId() 不要 int サーバー固有IDを取得
getAutoSave() 不要 bool 自動保存が有効か
setAutoSave($value) bool $value なし 自動保存を有効にする
getGamemode() 不要 int ゲームモードを取得
getDefaultGamemode() 不要 int デフォルトのゲームモードを取得
getDifficulty() 不要 int 難易度を取得
getSpawnRadius() 不要 int スポーン地点周りの保護
getAllowFlight() 不要 bool 飛行チートが許可されているか
getOfflinePlayer($name) string $name なし オフラインプレイヤーのオブジェクトを取得
getOfflinePlayerData($name) string $name なし オフラインプレイヤーのデーターを取得
matchPlayer($partialName) string $partialName なし Player[]
getOps() 不要 array OPを取得
addOp($name) string $name なし OPを追加
removeOp($name) string $name なし OPから除外
addWhitelist($name) string $name なし Whitelistに追加
removeWhitelist($name) string $name なし Whitelistから削除
reloadWhitelist() 不要 なし Whitelistを再読み込み
・Broadcast
broadcastMessage($message) string $message なし 全員にメッセージを送信
broadcastTip($tip) string $tip なし 全員にtipを送信
broadcastPopup($popup) string $popup なし 全員にpopupを送信
