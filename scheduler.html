Scheduler関係

説明
・USE文
use pocketmine\scheduler\PluginTask; PluginTaskの使用のために追加
use pocketmine\scheduler\CallbackTask; CallbackTaskの作成のために追加
use pocketmine\scheduler\Task; Taskの使用のために追加
・新規CallbackTaskの作成
new CallbackTask([$this, '実行したい関数名'], []) 新規CallbackTaskの作成
関数名 パラメータ 説明
scheduleTask($task) Task $task Taskを作成します
scheduleAsyncTask($task) AsyncTask $task 非同期処理のTaskを作成します
scheduleDelayedTask($task, $delay) Task $task, $delay 指定した時間後(40ならば2秒後)に実行されます
scheduleRepeatingTask($task, $period) Task $task, $period 指定した時間(40ならば2秒ごと)ごとに繰り返します
scheduleDelayedRepeatingTask($task, $delay, $period) Task $task, $delay, $period $delay後に$periodごとに繰り返します
cancelTask($taskId) int $taskId 指定したTaskをキャンセルします。
cancelTasks($plugin) Plugin $plugin 指定したPluginからのTaskをすべてキャンセルします。
cancelAllTasks() 不要 すべてのTaskをキャンセルします。
